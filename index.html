Portfolio

Gourmazin

Link to websiteGourmazin are a local UK business that provide gourmet food for festivals, weddings and private parties. This project was made using Jekyll a “blog-aware static-site” program which allowed me to make the website in HTML whilst also being able to provide the blogging features other CMS have.

I also used Sass and Gulp to decrease development time as well as to make the project scalable. By using Sass I created multiple variables with colours and font families that allowed me to resuse code without having to rewrite it. I used Gulp to prosses the sass as well as minimize files and add prefixes so that the website looks the same on every web browser.

Luke Flisher

Link to imageLuke is an independent photographer who provides his services for weddings and other photoshoots.

For this project we agreed it would require a portfolio where you can easily add new content as well as blog posts thus I used Wordpress. This was my first website and he has since changed the design.

Through Wordpress I used the blog feature for blog posts as well as for the portfolio. I have since learned that that isn’t very practical however back then that was what I thought off.

I also used Sass and Gulp. Sass for the much cleaner syntax and reusability and Gulp for compiling the files and providing helpful error messages.

Exilian FX

Link to imageExilian FX is an effects make up company and this project similar to Flisher, was done in Wordpress.

In this project I explored multiple ways to structure the blog posts but in the end I went for the masonry look having the stack on top of each other. This effect made it very easy to make the design responsive.

Moreover I wanted to create a fluid experience for the user so I usedangular in order to create a “Load more” button for blog posts that didn’t cause a full page reload.
